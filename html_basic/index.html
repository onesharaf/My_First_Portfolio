Welcome to My First Webpage

This is the main page of my new website. I am learning how to structure content using HTML, which is the skeleton of every webpage. It's exciting to see how different tags create different elements on the page.

About This Page

On this page, we are exploring several key HTML tags. We have paragraphs, like this one, and we also have different levels of headings to organize our topics clearly. Learning to nest tags is also important, like making an image clickable.

Key Elements Added

So far, we have added headings of levels 1, 2, and 3. We have also included several paragraphs of text to act as content. The next element is a clickable image, which combines an anchor tag (for the link) and an image tag.

  • The image above is the ALX logo. I've wrapped it in an anchor tag (a link) so that if you click on it, it will take you to the official ALX website. I also added an 'alt' attribute to the image, which is important for accessibility as it describes the image for screen readers.
